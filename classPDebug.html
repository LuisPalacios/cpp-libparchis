Description

This class shows debug messages on screen (terminal). It should be instantiated by each object that wants to show messages. Each object will use its own Debug object, but all will share the "debug level".

PDebug usage:
MyClass.h
class MyClass {
:
private:
PDebug dbg;
MyClass.cpp
// Constructor
PDebug::PDebug ( ) {
dbg.setCallerName ( "MyClass" );
dbg.setDebug(2);
}
