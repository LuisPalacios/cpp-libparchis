Description

This class represents each of the Spaces in the Parchis game. It holds information about the index and number, color, role, orientation (horzontal, or vertical), number of pawns at the space, etc. It also has several methods to query the space if a pawn can be placed into it, if there is an enemy, if a pawn can walk through, etc.
