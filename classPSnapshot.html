Description

Class that represents the overall status of the Game and can be used by Robots to make its calculations.

This class represents the Snapshot of Pawn in the current turn, for example, can eat, is eatable, can move, its coleagues are in danger, etc... It is used by Game to inform the Robots (at every turn) about the actual rationale, situation and conditions of each Pawn at the board in the current turn. This will allow the robots to make their decisions
