PPawn *  new_pawn,

Calculate if it's possible to move forward the passed Pawn by 'new_forward' spaces. This method will go through each Space in the path and verifying if the Pawn can pass through them.

Parameters
boardis the pointer to the Board
new_pawnis the Pawn for which I will calculate
new_forwardis the number of spaces wanted to move forward
Returns
Space * to the space where the pawn is to be moved

This method is not only used by the Rules for normal playing but also by the Robots for its analysis.
