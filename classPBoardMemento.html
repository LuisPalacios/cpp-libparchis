Description

Stores/Restores internal state of the Board object, it's based on the Memento design patter. Good reference (spanish) www.freewebz.com/amanecer/personal/papers/paper.memento.pdf

A memento is an object that stores a snapshot of the internal state of another object. It can be leveraged to support multi-level undo.

Originator Parchis.class (Creates the memento object) Memento ParchisMemento.class (Stores/Restores internal state of a Player object) CareTaker Game.cpp (Calls parchi's createMemento/setMemento to control the undo/redo's)
